# LLM абетка

Навчальні матеріали про великі мовні моделі для звичайних людей — без формул, простою мовою, з інтерактивними прикладами.

## Міні-курси

### [Як працюють LLM-трансформери](https://nomadic.github.io/llm-abetka/llm-minicourse.html)

Інтерактивний мінікурс з 23 розділів — від токенів до агентів. Охоплює:

- Токенізація та ембединги — як комп'ютер «читає» текст
- Механізм уваги (Attention) та архітектура трансформера
- Навчання моделі — функція втрат, градієнтний спуск, backpropagation
- SFT, RLHF, LoRA — як модель стає асистентом
- Практичні аспекти — KV-кеш, квантизація, контекстне вікно, scaling laws
- Використання — prompt engineering, галюцинації, RAG, агенти

## Для кого

Для всіх, хто хоче зрозуміти як працюють ChatGPT, Claude та інші LLM — без потреби у технічній освіті.
